How we can Add CSS in different Ways

Inline CSS

<h1 style="background-color: color we want to add;"> The Data </h1>

Internal CSS

<DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Three Ways to Add CSS </title>
<style>
div{
background-color:red;
}
</style>

In this way we can add Internal CSS in HTMl File only.

External CSS

<link rel="stylesheet" type="text/css" href="style.css">

First we Will Create a File named style.css and we have to link that file in HTMl and we can write our CSS in that File as a External CSS
